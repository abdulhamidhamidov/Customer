Проект 1: Система аренды велосипедов или самокатов
Описание: Система для управления процессом аренды велосипедов или самокатов. Каждое задание автономное и не зависит от других.

Задание 1: Управление транспортными средствами
- Сущность: Vehicle
  - Поля:
    - Id (int) — Уникальный идентификатор.
    - Type (string) — Тип (например, "Велосипед", "Электросамокат").
    - Brand (string) — Бренд.
    - Model (string) — Модель.
    - ProductionYear (int) — Год выпуска.
    - RentalPricePerHour (decimal) — Цена аренды за час.
    - MaxWeightCapacity (decimal) — Максимальная грузоподъемность (в кг).
    - BatteryCapacity (decimal?) — Емкость батареи (в кВт·ч, для электротранспорта).
    - IsAvailable (bool) — Доступен ли для аренды.
- Интерфейс: IVehicleService
  - Методы:
    - Добавить транспортное средство.
    - Обновить информацию о транспорте.
    - Удалить транспортное средство.
    - Получить список всех транспортных средств.

Задание 2: Управление клиентами
- Сущность: Customer
  - Поля:
    - Id (int) — Уникальный идентификатор.
    - FullName (string) — Полное имя.
    - PhoneNumber (string) — Номер телефона.
    - Email (string) — Электронная почта.
    - DateOfBirth (DateTime) — Дата рождения.
    - MembershipType (string) — Тип членства (например, "Стандарт", "Премиум").
    - RegistrationDate (DateTime) — Дата регистрации.
    - IsActive (bool) — Активен ли клиент.
- Интерфейс: ICustomerService
  - Методы:
    - Добавить клиента.
    - Обновить информацию о клиенте.
    - Удалить клиента.
    - Получить список всех клиентов.

 Задание 4: Управление промокодами
- Сущность: PromoCode
  - Поля:
    - Id (int) — Уникальный идентификатор.
    - Code (string) — Код промокода.
    - DiscountPercentage (decimal) — Процент скидки.
    - ExpiryDate (DateTime) — Дата окончания действия.
    - MinimumRentalAmount (decimal) — Минимальная сумма аренды для активации.
    - IsActive (bool) — Активен ли промокод.
- Интерфейс: IPromoCodeService
  - Методы:
    - Добавить промокод.
    - Обновить информацию о промокоде.
    - Удалить промокод.
    - Получить список всех промокодов.
